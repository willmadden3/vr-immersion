# Hardware

Physical hardware designs, schematics, and bills of materials.

## Directory Structure

- `schematics/` - Electrical circuit diagrams
- `bom/` - Bills of materials with part numbers and sources
- `cad/` - 3D models for enclosures, mounts, custom parts

## Subsystems

### VR Smell
- Pneumatic circuit (pump → valves → chambers → output)
- Electrical circuit (Arduino → transistors → valves)
- Smell chamber construction

### Climate Control (Future)
- Fan array for wind simulation
- IR panels for radiant heat
- Cooling system for cave environment

### Physical Props (Future)
- Kayak seat with paddle mechanism
- Bench seating
- Walking pad integration
